# Documentation & Help Resources

- **Built-in**: `help <command>`, `search <keyword>`, `db <command>` (dialog box)
- **Statalist** (official user forum): statalist.org
  — search before posting; include Stata version, full error message, and a reproducible example
- **SSC packages**: `ssc describe <pkg>` to preview; full listing at econpapers.repec.org
- **reghdfe**: `help reghdfe` or github.com/sergiocorreia/reghdfe
- **csdid**: github.com/friosavila/csdid (includes vignette and FAQs)
- **did2s**: github.com/kylebutts/did2s_stata
- **G&S guide**: `/projectnb/econdept/snitkin/resources/shapiro_coding_guide.md`
- **Stata manuals**: `help` links to PDF manual entries; also available at stata.com/manuals/

When debugging an error, always check the `.log` file first — Stata prints the exact command that failed and the full error message there, which is more informative than the console display.
